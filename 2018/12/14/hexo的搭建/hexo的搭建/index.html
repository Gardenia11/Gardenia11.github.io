title: hexo的搭建
date: 2018-12-14 00:18:51

tags: hexo

准备：安装node.js , git ；创建github账号；

1、github添加仓库

2、安装hexo

打开gitbash输入命令

$ sudo npm install -g hexo-cli

初始化

创建一个文件夹存放博客，cd进入文件夹输入命令

$ hexo init

4、部署到GitHub

安装git部署插件，输入命令

$ npm install hexo-deployer-git --save

修改站点配置文件（_config.yml)（冒号后都有一个空格）

deploy：

type: git
repo: (github地址）
branch: master

url:

github地址

后台输入 hexo g 和 hexo d

说明：

hexo s等同于hexo serve生成本地预览

hexo clean清除本地生成的文件

hexo g 等同于hexo generate生成最终的静态html文件

hexo d等同于hexo deploy部署到github
